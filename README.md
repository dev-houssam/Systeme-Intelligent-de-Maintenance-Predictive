# Système Intelligent de Maintenance Prédictive
Système Intelligent de Maintenance Prédictive

Ce document présente le plan complet du projet de maintenance prédictive industrielle. L’objectif est 
de concevoir un système capable de surveiller les capteurs des machines, détecter les anomalies, 
prédire les pannes à l’aide de modèles d’intelligence artificielle et alerter automatiquement le technicien 
via une interface web et un système de notification.
